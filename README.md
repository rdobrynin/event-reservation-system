# Event Registration System

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Tests

### Run tests

```bash
# unit tests
$ pnpm run test
[test](test)
# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
